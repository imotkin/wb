# L2.6

Что выведет программа?

Объяснить поведение срезов при передаче их в функцию.

```go
package main

import (
    "fmt"
)

func main() {
    var s = []string{"1", "2", "3"}
    modifySlice(s)
    fmt.Println(s)
}

func modifySlice(i []string) {
    i[0] = "3"
    i = append(i, "4")
    i[1] = "5"
    i = append(i, "6")
}
```
---

## Вывод

```
[3 2 3]
```

## Объяснение

При передаче в функцию происходит копирование переменной `i`, то есть можно изменить элементы слайса, но если `append` выделит новый массив для слайса при переполнении текущего, то он будет сохранён в копии переменной, которая передана в функцию. Именно поэтому в данном примере обновится первый элемент слайса, но при выполнении `append` для элемента `4` выделяется новый массив и записывается в переменной `i`, то есть после `append` изменится копия `i`, но не та переменная, что была передана в функцию.
