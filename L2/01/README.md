# L2.1

Что выведет программа?

Объясните вывод программы.
```go
package main

import "fmt"

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```
---

## Вывод

```
[77 78 79]
```

## Объяснение

В данном случае выводятся значения с первого по третий из массива `а`, полученные с помощью выражения для слайса `a[1:4]`. Длина полученного слайса `b` будет равна 3 (4-1), а ёмкость это 4 (5-1). Так как внутри слайс ссылается на массив `a`, то любое изменение массива `a` или слайса `b` повлияет и на другую переменную.
