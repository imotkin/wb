# L2.2

Что выведет программа?

Объясните вывод программы.
```go
package main

import "fmt"

func test() (x int) {
    defer func() {
        x++
    }()
    x = 1
    return
}

func anotherTest() int {
    var x int
    defer func() {
        x++
    }()
    x = 1
    return x
}

func main() {
    fmt.Println(test())
    fmt.Println(anotherTest())
}
```
---

## Вывод
Данная программа выведет следующее:
```
2
1
```

## Объяснение 
Данный пример показывает особенности работы выражения `defer`. Функция `test` имеет именованное значение `x` для возврата и при запуске `defer` оно увеличивается на один и функия возвращает обновлённое значение `2`. Однако для локальной переменной происходит ситуация иначе, так как при возврате из функции значение x уже было скопировано, как 1, и вызов `defer` влияет лишь на локальную переменную, а возвращаемая копия для `x`, равная `1`, остаётся без изменений.
